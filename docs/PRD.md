## 1. 제품 개요

### Epiphany — 제품의 출발점

> 서울 헬리오시티 한 단지에서 연 **387,000톤**의 깨끗한 지하수가 하수도로 그냥 흘러간다.
> 하수도요금 50% 감면 조례가 시행된 지 4년이 지났지만, 2024년 건축물 미사용률은 여전히 **92.1%**.
>
> 이건 수요가 없어서가 아니라, **얼마를 아낄 수 있는지 계산해주는 도구가 없어서** 안 풀리는 문제다.

### 한 줄 정의

서울시 공공데이터 기반 AI가 건물별 유출지하수 절감액을 자동 계산하고,
설비 투자 ROI를 분석하며, ESG 탄소절감 리포트를 자동 생성하는 **B2B SaaS**.

### 핵심 숫자 (2024, 서울 열린데이터광장 직접 집계)

| 구분 | 총 발생량 | 미활용 방류 | 미사용률 |
|------|-----------|-------------|----------|
| 건축물 | 1,806만 톤 | 3,318만 톤 | **92.1%** |
| 지하철 | 4,681만 톤 | 1,931만 톤 | 20.7% |
| 전력통신구 | 892만 톤 | 330만 톤 | 18.4% |
| **합계** | **7,379만 톤** | **5,579만 톤** | **75.6%** |

→ 일 평균 15만 3천 톤, 건물주 절감 가능 **연 약 223억원** (추산).

## 2. 왜 지금까지 안 풀렸나

### 2.1 건물주 3가지 장벽

1. **경제성 계산 불가** — 감면 조례가 있다는 건 알지만, 우리 건물에서 실제 얼마를 절감하는지 계산 도구 없음
2. **설비 투자 근거 없음** — 발생량 예측·ROI 분석이 없어 투자 결정 불가
3. **ESG 리포트 작성 불가** — GRI 형식 정량 데이터를 수작업으로 만들어야 함

### 2.2 카운터-아규먼트 응답

> **"한국은 물이 풍부해서 굳이 쓸 필요가 없다."**

| 축 | 근거 |
|----|------|
| **비용** | 하수도요금 50% 감면 = 즉각 절감. 버리는 것도 비용 |
| **ESG 의무화** | 금융위 2026 대형 상장사 ESG 공시 의무화 → 환경 데이터 수요 급증 |
| **도시 물순환** | 하수도 방류 = 열섬·지반침하 유발 |
| **탄소 절감** | 정수→공급→재정수 펌핑 에너지 절감 → CO₂ KPI 정량화 가능 |
| **시장 촉매** | 지하공간 개발 확대 (2020 7,167만 톤 → 2022 7,845만 톤) |

## 7. AI/ML 명세 (창업 부문 핵심 어필)

> **솔직한 자기제한**: 1인 D-3 제약에서 "도메인 베이스라인 + ML 잔차 보정" 패턴 채택.
> NeurIPS _Climate Change AI_ 워크숍에서 정식 인정되는 접근 — 도메인 신뢰도와 정확도를 동시에 확보.

### 7.1 발생량 예측 모델

**과업**: 건물별 30일 재이용 가능량 예측

**입력 변수**

- 환경: 최근 30일 강수량, 월별 지하수위 변동 (기상·지하수 관측)
- 건물 특성: 지하층 수, 연면적, 준공연도, 발생량 이력 (건축물 유출지하수 현황)
- 계절성: 월·분기 더미 변수

**모델 트랙 (앙상블)**

1. **수문학 베이스라인** — 기준증발산량 ET₀ (Penman-Monteith) × 작물계수 × 면적
2. **LightGBM Residual** — 베이스라인 잔차를 ML로 보정
3. _(Stretch)_ XGBoost 2.0 + Prophet 1.1 앙상블 — 시계열 예측 정확도 향상

**검증**

- Walk-forward CV (월 단위 시차 검증)
- 메트릭: MAPE < 15% (전체), MAPE < 35% (Demo)
- 설명력: SHAP 값으로 강수량·계절성·지하층 수 기여도 시각화

### 7.2 ROI 자동 계산 로직

| 항목 | 계산식 |
|------|--------|
| 하수도요금 절감 | 재이용량(톤/년) × 400원/톤 × 50% (하수도요금 감면 조례, 2022~) |
| 수도요금 절감 | 재이용량 × 수도단가 (용도별) |
| CO₂ 절감량 | 재이용량 × 0.4 kWh/톤 × 0.4594 tCO₂eq/MWh (한국에너지공단 배출계수) |
| PBP | 설비 투자비 ÷ 연간 절감액 (단순 투자회수 기간) |

**헬리오시티 사례 시뮬레이션**

| 항목 | 수치 | 산출 근거 |
|------|------|-----------|
| 연간 발생량 | 387,000 톤/년 | 서울시 컨설팅 결과 (2021) |
| 하수도요금 절감 | 약 7,740만원/년 | 387천 톤 × 400원 × 50% |
| CO₂ 절감량 | 약 71.1 tCO₂eq/년 | × 0.4 kWh/톤 × 0.4594 tCO₂eq/MWh |

### 7.3 공급-수요 매칭 최적화

**문제**: 가중 이분그래프 매칭 with capacity (Capacitated Bipartite Assignment)

**제약**

- Σ_j x_ij ≤ S_i (공급량 한도)
- Σ_i x_ij ≤ D_j (수요량 한도)
- x_ij = 0 if dist(i,j) > R (반경 제약)
- x_ij = 0 if 수질등급(i) ∉ 허용범위(j)

**알고리즘**: PuLP + CBC (오픈소스 ILP), N < 1000

## 9. 핵심 기능 요구사항

### FR-01. 지도 기반 시각화·건물 검색

| 레이어 | 표현 방식 | 표시 정보 |
|--------|-----------|-----------|
| 공급처 (건물) | 파란 핀 마커 (크기 = 일발생량) | 건물명, 톤/일, 수질 등급, 신고상태 |
| 수요처 (공원·도로) | 주황 원형 마커 (크기 = 예측수요) | 유형, 예측 일수요량 |
| 매칭 경로 | 녹색 선 | 공급처→수요처, 거리, 배분량 |

**인터랙션**

- 건물명·주소 검색 → 해당 건물 포커스 + 현황 사이드패널
- 반경 슬라이더 (500m / 1km / 2km)
- 마커 클릭 → 상세 패널 (발생량, 수질, 신고상태)

### FR-02. AI 발생량 예측

- 건물별 30일 재이용 가능량 예측
- 강수량·계절성·지하층 수 각 요인별 기여도 차트 (SHAP)

### FR-03. 절감액·ROI·PBP 자동 계산

- 재이용 목적 선택 (냉방·청소·조경) → 절감액·PBP 자동 산출
- 월별 절감액 시각화

### FR-04. Epiphany 대시보드

상단 카운터 카드 4개:

- "오늘도 서울에서 **\_\_\_톤**의 유출지하수가 하수도로 흘러갔습니다"
- "매칭 시 절감 가능 수도요금: **연 \_\_\_억원**"
- "탄소 절감: **\_\_\_tCO₂eq/년**"
- "현재 활용률: 0% → 시스템 적용 시 **\_\_\_%**"

### FR-05. 수요처 매칭 지도

- 공원·도로 수요처 자동 매핑
- 반경 N km 내 최적 매칭 (PuLP ILP)
- (Phase 2) 생활인구 기반 인근 수요처 밀도 추천

### FR-06. ESG 리포트 자동 생성 (Phase 2)

- CO₂ 절감량 집계
- GRI 303·305 양식 자동 매핑
- PDF 출력 (ReportLab)

## 11. 수익 모델 (B2B 퍼널)

초기 구독 저항을 낮추기 위해 '무료 진단 → 유료 컨설팅 → SaaS 구독' 단계적 전환 전략.

| 단계 | 모델 | 대상 | 단가 |
|------|------|------|------|
| **1단계 (무료)** | 무료 절감액 계산기 | 모든 건물주 | 무료 (리드 획득) |
| **2단계 (유료 진단)** | 현장 진단·설비 도입 타당성 컨설팅 | 하루 500톤+ 발생 건물 | 300~500만원/건 |
| **3단계 (SaaS)** | 월간 구독 — 대시보드 + 리포트 | 컨설팅 완료 고객 | 월 10~100만원 |
| **추가 수익** | 하수도 감면 신청 대행 수수료 | 감면 수혜 건물 | 절감액의 10% |

**B2G 병행 (2순위)**

| 우선순위 | 수익원 | 대상 | 단가 |
|---------|--------|------|------|
| 1 | 데이터 분석 용역 / 연간 라이선스 | 서울시·광역시 환경·상수도 부서 | 연 3,000만~1.5억원 |
| 2 | 분기·연간 리포트 발간 | 환경부·구청 | 연 500~2,000만원 |

### 매출 시나리오

| 연도 | 모델 | 고객 수 | 연간 예상 매출 |
|------|------|---------|---------------|
| Year 1 | 컨설팅 위주 | 컨설팅 10건 + SaaS 30개소 | 1.5억~3억원 |
| Year 2 | SaaS 확장 | SaaS 150개소 + 컨설팅 30건 | 6억~10억원 |
| Year 3 | 플랫폼화 | SaaS 500개소 + 타 지자체 확장 | 20억원+ |
